API REST Interactive

Testez tous les endpoints directement depuis votre navigateur

Clients

  • GET /clients - Liste des clients
  • POST /clients - Créer un client
  • GET /clients/{id} - Détails d'un client
  • PUT /clients/{id} - Modifier un client
  • DELETE /clients/{id} - Supprimer un client

Comptes

  • GET /comptes - Liste des comptes
  • POST /create - Créer un compte
  • GET /comptes/{id} - Détails d'un compte
  • PUT /comptes/{id} - Modifier un compte
  • DELETE /comptes/{id} - Supprimer un compte
  • GET /solde?compteId={id} - Consulter le solde
  • POST /transfer - Effectuer un transfert
